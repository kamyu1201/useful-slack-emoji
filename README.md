# useful-slack-emoji
新しく入ったslackとかを自分のemojiで侵略するための絵文字集である！

# ルール
* 汎用的にするため、個人名はいれない
* できるだけ、同じ用途のものはひとつずつにする
* 文字の絵文字のときは、名前そのままのファイル名にする
* だれかが嫌な気持ちになるようなものは避ける
